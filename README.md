# Chatrobot

Chatrobot是学校的企业微信平台关于招生的对话机器人。

组长：李世鲜

组员：白壹方 贾新宇

小组分工： 李世鲜负责对话机器人的搭建和问答对整理，白壹方负责API对接方面的工作和DJango框架搭建，贾新宇负责后台搭建。

程序所用到的是Python的Django框架。

目前进度:已在企业微信的机器人中实现通过问答对的方式实现简易的“对话”。
